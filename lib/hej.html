• main#between_strict(n1, n2, n3)
    Public: Decides if a number is strictly between two other numbers

n1 - The number that is said to be strictly between
n2 - The first number that n1 is between
n3 - The second number that n1 is between

Examples

  between_strict(3, 2, 1)
  # => true

Returns true or false depending on if the statement is true
